Gemini Event Creator

Turn any text into a calendar event

Service

? How to use

  • 1
    On any page, highlight text that sounds like an event (meeting, call, flight, etc.).
  • 2
    A small purple bubble appears at the bottom-right: "Create calendar event".
  • 3
    Choose Quick Create to instantly save, or Edit & Create to adjust title, time, location, color, and reminders first.
  • 4
    After you create the event, that bubble closes and a short confirmation shows in the same corner.

★ What you can edit

Title Start / End time Description Location
Color label Reminder time

Color and reminders work like Google Calendar: pick a color dot, then pick when you want to be notified (30 min before, 1 hr before, etc.).

i Good to know

• Events are created using your calendar access from the extension's background script.

• If you don’t see the purple bubble after highlighting text, the page may block content scripts (for example, some Chrome internal pages or browser UI). Try on a normal webpage.

• You’ll also get a small confirmation toast in that same corner after each create.
